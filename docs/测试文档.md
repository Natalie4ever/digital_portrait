## 七、数据校验规则汇总

| 字段 | 规则 | 错误提示 |
|------|------|----------|
| EHR 号 | 恰好 7 位数字，允许前导零 | EHR 号必须为 7 位数字 |
| 证件号码 | 18 位，校验码正确，出生日期合法 | 证件号码必须为18位 / 格式不正确 / 校验失败 |
| 手机号 | 1 开头 11 位 | 手机号格式不正确 |
| 电话 | 7–20 位数字、横线、空格等 | 电话格式不正确 |
| 日期 | YYYY-MM-DD、YYYY/MM/DD、YYYYMMDD | 日期格式不正确，请使用 YYYY-MM-DD |
