## 七、数据校验规则汇总

| 字段 | 规则 | 错误提示 |
|------|------|----------|
| EHR 号 | 恰好 7 位数字，允许前导零 | EHR 号必须为 7 位数字 |
| 证件号码 | 18 位，校验码正确，出生日期合法 | 证件号码必须为18位 / 格式不正确 / 校验失败 |
| 手机号 | 1 开头 11 位 | 手机号格式不正确 |
| 电话 | 7–20 位数字、横线、空格等 | 电话格式不正确 |
| 日期 | YYYY-MM-DD、YYYY/MM/DD、YYYYMMDD | 日期格式不正确，请使用 YYYY-MM-DD |

| HV-027 | 组长无组别新建 | 同上 | 访问新建并尝试提交 | 提示「请联系管理分配组别」或等价错误 |
| HV-028 | 新建时选择非本组成员 | 通过接口或异常路径提交他组 EHR | 提交创建请求 | 返回 403「只能对本组成员进行家访」 |
| HV-029 | 查看不存在记录 | 已登录 | 访问 `/home-visits/{不存在id}` | 显示「家访记录不存在」 |
| HV-030 | 会话过期处理 | 已登录后保持无操作超过 1 小时 | 再执行任意接口请求 | 前端清除 token 并跳转登录页 |
| HV-031 | 管理员编辑接口兜底 | 管理员登录且存在记录 | 通过接口调用 `PUT /api/home-visits/{id}` 修改数据 | 返回 403，提示管理员不可编辑 |
| HV-032 | 普通用户查看他人详情越权 | 普通用户登录且存在他人记录 | 通过地址栏或接口访问他人记录详情 | 返回 403，无权限查看 |
| HV-033 | 组长查看非本组详情越权 | 组长登录且存在非本组记录 | 通过地址栏或接口访问非本组记录详情 | 返回 403，无权限查看 |
| HV-034 | 非法 EHR 筛选行为 | 家访列表存在多条记录 | 在 EHR 筛选输入非法格式（非 7 位数字）后查询 | 请求可成功返回；该筛选条件被忽略（以当前实现为准） |
